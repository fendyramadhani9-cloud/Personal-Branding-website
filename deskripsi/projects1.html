AI WhatsApp Assistant — Mama Cookies

Project ini adalah asisten WhatsApp berbasis AI yang menggunakan model bahasa Qwen 7B melalui Ollama. Bot ini berfungsi sebagai kasir virtual Mama Cookies yang dapat menjawab pertanyaan pelanggan seperti informasi produk, harga cookies, rekomendasi menu, dan ketersediaan produk.

Sistem menggunakan LLM lokal sehingga dapat berjalan tanpa ketergantungan API cloud — lebih privat, lebih cepat, dan gratis digunakan.

Teknologi

JavaScript JSON Qwen2.5:7B WhatsApp Bot Ollama
